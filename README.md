# 1. 게임 Gungeon Heroes GDD

## 1.1. Gungeon Heroes는 로그라이트 액션 RPG 입니다.



```markdown
# Heading 1 (H1) - Main title

## Heading 2 (H2) - Section title

### Heading 3 (H3) - Subsection title
```

## Ordered Lists (순서가 있는 리스트)

```markdown
1. First item
2. Second item
3. Third item
```

## Unordered Lists (순서가 없는 리스트)

```markdown
- Item one
- Item two
- Item three
```

## Text Formatting (텍스트)

```markdown
**Bold text** - 굵게
_Italic text_ - 기울임
`Code` - 코드
~~Strikethrough~~ - 취소선
```

## Hyperlinks (하이퍼링크)

```markdown
[Link text](https://example.com)
```

## Images (이미지)

```markdown
![alt text](./image.png)
```
